# UEFA Club Competition Participants — 2010 to 2025

This document lists all clubs that participated in UEFA club competitions from 2010–11 to 2024–25, showing the seasons they appeared and the city attractions JSON file used by **MMM-iAmGoingThere** when that team is set as a destination.

**Competitions covered:**
- **UCL** — UEFA Champions League (Group / League Phase, 2010–11 to 2024–25)
- **UEL** — UEFA Europa League (Group / League Phase, 2010–11 to 2024–25)
- **UECL** — UEFA Europa Conference League (Group / League Phase, 2021–22 to 2024–25)

**Notes:**
- Clubs are sorted alphabetically by country, then by club name within each section.
- Where multiple clubs share a city the same JSON file is used (e.g. `london.json` for all London clubs).
- ⚠ **Displaced clubs:** Shakhtar Donetsk and Dynamo Kyiv played home matches at neutral venues from 2022 onwards due to the war in Ukraine. The venue city used is noted in the table.
- Russian clubs are excluded from UEFA competition from 2022–23 onwards (UEFA suspension).
- JSON files marked `*` were created as part of the 2010–2025 UEFA expansion project.
